    • Beach-Bot

    Introduction

    Beach-bot is a open source, Discord bot written using the discord.py written by Rapptz Danny, which serves as a template for all the newcomers to d.py, subsequently making development easy!

    You will be learning how to create a bot on Discord, then start developing it using the Discord API. The following are the pre-requiresites:

  • A PC most preferable
  • Good internet
  • A Discord account
  • Basic Python knowledge
  • Python environment installed on your machine

    • The following showcases what you will be learning by going through Beach-bot in depth. They are:

  • Syntax of discord.py
  • Hybrid and Slash commands
  • User and message context menu commands
  • Components, i.e., buttons, select menus and modals

    • Creating a bot

    Intents are permissions given to the bot to access certain information of the API in order to reduce resource usage.

    Step 1

    We would be creating an application in the developer portal of Discord. Click here to go to the portal. Click on the "New Application" button which is present on the top right hand side corner.

    Step 2

    Now type in a name for the application, ensure the name is unique and not common as there would be many other bots with the same name.

    Step 3

    Under the "Settings" category on the left hand side, click on the "Bot" section.

    Step 4

    Click on the "Add Bot" button which is on the right hand side corner. You will get a confirmation modal in which you will click "Yes" on it. If you get an error confirming, it means the name you had given for the application is way to common and a new bot cannot be created with the same name.

    If you got no error, then pat yourself on the back! You just created your first ever bot on Discord ðŸ˜Ž

    Step 5

    Click on "Click to Reveal Token" to reveal the secret token of your bot and copy it. Please ensure that this token is not shared with anyone. If anyone gets any access to this token, you will loose complete access to the bot as they will be able to run an instance of your bot and damage everything.

    Step 6

    To invite your bot to a server, go to the "OAuth2" category and select the "General" option. Under that, click on the dropdown under "Authorization Method" and choose "In-app Authorization".

    Select both "bot" and "application.commands" scopes. The bot scope enables Discord to treat the application as a bot and the applications.commands scope enables the bot to utilise application commands. Select all the permissions you want the bot to have by default.

    The "In-app Authorization" allows users to invite bots into other servers from the bot profile itself. Save the changes you've made so far.

    Step 7

    Once again, under the "OAuth2" category, now select the URL Generator and click on the "bot" scope and choose your permissions. A url would've been generated below, copy paste the url in a different tab to invite the bot.

    Step 8

    We now need to enable few intents for the bot to work in general. In the "Bot" section, enable "Server Member" (access information of server members), "Message Content" (access information from messages), and "Precense Intent" (access precense information of users).

    On enabling this, you will be able to access the information accordingly and execute commands in ease. However, upon reaching 100+ guilds, you would have to get your bot verified and send Discord a proper reason in order to continue access the information.

    What's Next?

    You have now created a bot and invited it into your server! All you now need is to code all the features you want!
