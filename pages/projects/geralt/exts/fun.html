    • Fun Cog

    This cog is filled with commands that induce fun into your guild (aka server). Starting all the from reducing your stress, to being a reason for an entire war to be held. You've stepped right up, they're all here.

    If a command is suffixed with "- G", it indicates that it is a group command which comprises of two or more sub-commands.

    "<>" symbols mean the particular argument is required | "[]" symbols mean the particular argument is optional.You don't have to type out the symbols when invoking the command.

    Can be used 3 times every 5.0 seconds per user.

    click - G

    .gclick

    cl | clock

    Just click the button!

    Can be used 2 times every 5.0 seconds per user.

    .gclick start

    Nil

    --size Sends that many number of buttons to click. Limited to 10 👍

    Can be used 2 times every 10.0 seconds per user.

    Can be used 3 times every 5.0 seconds per user.
