    • Tag Cog

    Tags are a simple way to store a certain text and retrieve them later on whenever you need it. Suppose you're a support staff for any product, and people keep questioning the same boring question. Besides typing the answer each and every time, you simply invoke the command which invokes the tag! Or you could use it troll your friends too haha.

    The entire cog consists of one single parent command with 9 sub commands present. So you will now be seeing them one by one.

    "<>" symbols mean the particular argument is required | "[]" symbols mean the particular argument is optional.You don't have to type out the symbols when invoking the command.

    tag delete

    .gtag delete

    tag_name

    remove

    Delete a tag you own in a guild.

    Can be used 5 times every 5.0 seconds per user.

    tag import

    .gtag import

    flag

    Nil

    Import tags from other guilds.

    --tag ID of the tag | --guild ID of the guild

    .gtag import [--tag <int> --guild <int>]

    Can be used 5 times every 5.0 seconds per user.

    tag make

    .gtag make

    add

    Make a tag.

    Can be used 5 times every 5.0 seconds per user.

    tag list

    .gtag list

    user

    l

    Get a list of all tags owned by a user in a guild. Defaults user to you.

    Can be used 5 times every 5.0 seonds per user.

    tag raw

    .gtag raw

    tag_name

    r

    See the tag without any markdown present.

    Can be used 5 times every 5.0 seonds per user.

    tag transfer

    .gtag transfer

    tag_id

    user

    Nil

    Transfer a tag you own to another user in a guild.

    Can be used 5 times every 5.0 seonds per user.

    tag all

    .gtag all

    Nil

    Get a list of all tags present in a guild.

    Can be used 5 times every 5.0 seonds per user.

    tag edit

    .gtag edit

    tag_id

    edited_content

    Nil

    Edit a tags content which you own in a guild.

    Can be used 5 times every 5.0 seonds per user.

    tag info

    .gtag info

    tag_name

    Nil

    Get entire details of a tag in a guild.

    Can be used 5 times every 5.0 seonds per user.
