    • Guild Cog

    The Guild cog provides the best tools for server management. Ticketing services, verification controls, customisable prefixes, and many more! This makes server administration smooth as butter :D

    All the commands are suffixed with "- G", it indicates that it is a group command which comprises of two or more sub-commands.

    "<>" symbols mean the particular argument is required | "[]" symbols mean the particular argument is optional.You don't have to type out the symbols when invoking the command.

    emote - G

    .gemote

    em | emotes

    Group commands for emotes present in a guild.

    .gemote delete

    emote=None

    del

    Delete an emote.

    .gemote info

    emote=None

    Nil

    Get info on an emote.

    .gemote rename

    emote=None

    name

    rn

    Rename an emote.

    guild

    .gguild

    server

    Sub-commands for basic guild settings.

    Can be used 3 times every 20.0 seconds per user.

    .gguild auditlog

    al

    View your Audit Log!

    Can be used 1 every 30.0 seconds in each category.

    Can be used 1 time concurrently in a guild.

    .gguild convertemote

    ce

    Enable to convert emote urls to webhooks.

    .gguild snipe

    s

    Opt-in/out for sniping messages!

    prefix

    .gprefix

    p

    Prefix related commands.

    Can be used 3 times every 5.0 seconds per user.

    .gprefix add

    prefix

    a

    Add a custom prefix.

    Can be used 3 times every 5.0 seconds per user.

    .gprefix remove

    prefix

    del

    Remove a custom prefix.

    Can be used 3 times every 5.0 seconds per user.

    .gprefix request

    Nil

    Reset all the prefixes back to .g.

    stickers

    .gstickers

    stcki | sticker

    Get information on each sticker present!

    ticket

    .gticket

    tools | tt

    Ticketing system for support in a guild.

    Can be used 3 times every 20.0 seconds per user.

    Can be used 1 time concurrently in a guild.

    .gticket setup

    channel=None

    Nil

    Setup ticket system in a server.

    .gticket dismantle

    clear | delete | remove

    Disbales the ticket system in the guild.

    .gticket close

    ticket_id

    Nil

    Close a ticket.

    .gticket pending

    Nil

    Review pending tickets.

    .gticket status

    Nil

    Returns the ticket systems status for the guild.

    verification

    .gverification

    verify | vf

    Group commands for securing your guild.

    Can be used 3 times every 20.0 seconds per user.

    Can be used 1 time concurrently in a guild.

    .gverification setup

    channel

    Nil

    Secure your guild with verification.

    .gverification status

    >

    Nil

    Returns verification status for your guild.

    .gverification dismantle

    remove

    Dismantle verification system from your server.
